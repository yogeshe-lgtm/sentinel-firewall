# Logging System

## Overview

The Sentinel Firewall Logging System stores security events generated by the firewall.

## Features

- Event logging
- Timestamp recording
- Firewall decision storage
- CLI log viewing

## Log Location

logs/firewall.log


## Example

2026-06-30 08:50:12 | BLOCK | TCP | 192.168.1.20 -> 192.168.1.5
