# Packet Analysis Engine

## Overview

The Packet Analysis Engine extracts useful information from network packets captured by Sentinel Firewall.

## Features

- Source IP extraction
- Destination IP extraction
- Protocol identification
- Port extraction
- Packet size calculation
- Traffic statistics

## Supported Protocols

- TCP
- UDP
- ICMP

## Future Improvements

- Deep packet inspection
- Application detection
- Threat analysis
- Packet filtering integration
